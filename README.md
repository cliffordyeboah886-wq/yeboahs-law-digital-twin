# yeboahs-law-digital-twin
"Official Digital Twin implementation of Yeboah’s Law for Dynamic Material Separation. This simulator calculates the dimensionless efficiency parameter (η) using the Yeboah Constant (kY), providing predictive analysis for tool-material interactions across micro and macro scales."
